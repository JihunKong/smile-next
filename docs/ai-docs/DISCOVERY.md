# AI Documentation System - Discovery Guide

> **For AI Agents**: This file helps you discover and use the AI documentation system

## How AI Agents Find This System

AI agents in different IDEs can discover this system through:

1. **Root-level files**:
   - `AI_DOCS.md` - Root-level discovery file
   - `.cursorrules` - Cursor IDE configuration
   - `.aiderignore` - Aider AI assistant config
   - `.windsurf` - Windsurf IDE config
   - `.antigravity` - Antigravity IDE config

2. **README references**:
   - Main `README.md` links to AI documentation
   - `docs/ai-docs/README.md` - Full overview

3. **This file**:
   - Located in `docs/ai-docs/DISCOVERY.md`
   - Quick reference for discovery

## Quick Start

1. **Read**: `AI_AGENT_MEMORY.md` - Complete quick reference
2. **Rules**: `RULES.md` - Maintenance rules
3. **Registry**: `INDEX.md` - All documents
4. **Mapping**: `MAPPING.md` - Relationships

## When You Encounter Documentation

If you're modifying code or documentation:

1. Check if documentation exists in `docs/ai-docs/`
2. Read `AI_AGENT_MEMORY.md` for procedures
3. Follow `RULES.md` for maintenance
4. Update `INDEX.md` and `MAPPING.md` as needed

## Validation

After changes:
```bash
cd docs/ai-docs
node validate-docs.js
```
